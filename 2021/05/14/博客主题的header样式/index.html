博客主题的header样式

2021-05-14 css

UI框架用多了都快不会写css了，把这个博客的头部样式写了一下感觉挺好看的，不算100%还原吧，七八分真。

当时主题的header

可能以后换主题所以在这里录制一下当时主题的header样式做个对比

模仿主题的header效果图

思路

初始化为置顶

向下滑动文字由初始白色瞬间变成黑色，header由透明白色过渡到白色，header向上移动。

向上移动到置顶header向下移动，字体为白色，header由白色过渡到透明白色

和主题的不同

这个主题如果文章很长的话从最低端向上滑动100px左右的距离就会显示出header，我这里只有置顶才会显示header

主题是通过切换div的class样式，我是通过额外添加了一个div蒙版来进行的切换
